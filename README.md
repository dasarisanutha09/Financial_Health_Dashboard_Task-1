# Financial Health Dashboard

## Project Overview

The Financial Health Dashboard is an interactive Power BI dashboard developed to analyze business financial performance. The dashboard provides insights into revenue, expenses, profit trends, category-wise performance, and regional analysis.

This dashboard helps organizations monitor financial growth and make better business decisions using data visualization.

## Dataset Information

**Dataset Name:** Financial Health Dataset

The dataset contains financial information including:

* Transaction ID
* Date
* Category
* Region
* Revenue
* Expenses
* Profit
* Payment Mode

## Tools Used

* Power BI Desktop
* Excel Dataset
* Data Cleaning and Transformation
* Data Visualization
* Data Analysis

## Dashboard Features

### 1. Total Revenue KPI

Displays the total revenue generated by the organization.

### 2. Total Profit KPI

Shows the overall profit earned after expenses.

### 3. Profit Trend Analysis

Analyzes profit changes over different time periods.

### 4. Expense Distribution Analysis

Shows how expenses are distributed across different categories.

### 5. Region-wise Revenue Analysis

Compares business performance across different regions.

## Key Insights

* The dashboard provides a complete overview of financial performance.
* Revenue and profit trends can be easily monitored.
* Expense categories help identify major spending areas.
* Regional analysis helps identify high-performing regions.

The Financial Health Dashboard converts financial data into meaningful insights. It helps businesses analyze revenue, expenses, and profit patterns for effective financial planning.
